# NanoPuter
## Free Unlimited NanoBanana Image Generation.
---

## Instructions
- go to [https://paraspl01t.github.io/nanoputer/](https://paraspl01t.github.io/nanoputer/) In an **Incognito** Window.
- Click on RESET button and wait for popup to open. If popup doesn't open, check your browser settings.
- Solve the captcha and wait for popup to close.
- Enjoy unlimited image generation. If usage percentage reaches above 90%, Close the window and re open in another incognito tab.
